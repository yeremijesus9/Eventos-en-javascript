Métodos de Eventos en JavaScript

Abre la consola del navegador (F12) para ver los resultados de los ejemplos 6, 7, 8 y 9.

1. Etiqueta HTML (Método rápido)

El evento se define directamente en el atributo HTML onclick. **No recomendado** para código complejo.

2. Desde Script (Método/Función Directa)

Se asigna una función a la propiedad onclick del elemento en JavaScript. **Solo permite un manejador**.

3. Desde Script (addEventListener())

El método **recomendado**. Permite múltiples manejadores de eventos y más control.

4. Propagación (Burbujeo)

Muestra cómo un evento en el hijo se propaga al padre. Revisa la consola.

5. Con parámetros y función flecha

Uso de función flecha para llamar a una función con parámetros de forma concisa.

6. Objeto event

7. Datos de Elemento (id, nodeName, current)
